NP Bayes Paper is Out

Our paper on using nonparametric Bayes methods for biophysical time series is out in Biophysical Journal. I've been meaning to write an accessible introduction to the Dirichlet Process for a while now because I never found a great one while I was learning all this stuff. But I never got around to it. In any case, this was a really fun project working to incorporate these exciting new methods from the machine learning community into protein biophysics. We demonstrated some compelling potential here and there's definitely interesting future directions to go including extending beyond the Dirichlet Process. Gaussian Processes are on my mind lately and there's some very cool stuff to be done there.
